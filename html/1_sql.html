¿Qué es SQL?

Structured Query Language (SQL) es el lenguaje estándar utilizado para gestionar y consultar bases de datos relacionales.

Definición de SQL

SQL (Structured Query Language) es un lenguaje diseñado para crear, administrar y consultar información almacenada en bases de datos relacionales.

Permite organizar datos en tablas relacionadas mediante claves y reglas estructuradas, facilitando el almacenamiento y la recuperación eficiente de información.

Conceptos Fundamentales

Tablas

Son estructuras que almacenan información organizada. Cada tabla representa una entidad del sistema.

Filas

Representan registros individuales dentro de una tabla. Cada fila contiene datos específicos.

Columnas

Definen los atributos o características de los registros almacenados.

Claves Primarias y Foráneas

Clave Primaria (Primary Key)

Identifica de manera única cada registro de una tabla. No puede repetirse ni contener valores nulos.

Clave Foránea (Foreign Key)

Establece relaciones entre tablas, permitiendo conectar información de distintas entidades.

Importancia de los Esquemas Definidos

Un esquema define la estructura de la base de datos: tablas, columnas, tipos de datos y relaciones.

Gracias a los esquemas, los datos mantienen integridad, consistencia y organización, evitando errores y duplicidades.

Además, facilitan el mantenimiento y escalabilidad de los sistemas de información.

Ventajas y Desventajas de SQL

Ventajas

  • Alta integridad de datos.
  • Consultas potentes y flexibles.
  • Estándar ampliamente utilizado.
  • Soporte para relaciones complejas.
  • Seguridad y control de acceso.

Desventajas

  • Escalabilidad horizontal limitada.
  • Mayor rigidez estructural.
  • Diseño inicial más complejo.
  • Puede requerir optimización avanzada.

A continuación se muestra un ejemplo de un esquema relacional para un sistema de preguntas y respuestas: 


CREATE TABLE usuario (
    id_usuario INT PRIMARY KEY,
    nombre VARCHAR(100),
    correo VARCHAR(150)
);

CREATE TABLE pregunta (
    id_pregunta INT PRIMARY KEY,
    titulo VARCHAR(255),
    descripcion TEXT
);
